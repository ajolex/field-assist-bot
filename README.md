# Field Assist Bot

Field Assist Bot is a Discord teammate for field operations. It helps teams answer protocol questions, check case status, monitor progress, escalate blockers, and run approved automations.

## What it does

- Responds in Discord when tagged, with routing for common field workflows.
- Supports case checks, progress snapshots, and form-version updates.
- Handles issue triage with structured tracking and escalation.
- Posts scheduled reminders and nightly exception summaries.
- Runs a safe automation job for SurveyCTO + Stata through a whitelisted command.

## Who this is for

- Field officers and team leads who need quick answers in chat.
- Coordinators who monitor quality, status, and escalations.
- Data management staff running recurring download/processing routines.
